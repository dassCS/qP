# qP
more rust

cargo build --release
cargo install --path .

Encode and decode QP image files (.qp)

Usage: qp <COMMAND>

Commands:
  encode  Encode an image to QP format
  decode  Decode a QP file to an image
  help    Print this message or the help of the given subcommand(s)

Options:
  -h, --help  Print help
